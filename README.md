# Candy Land

A colorful and responsive candy shop website showcasing sweet treats and confections.

## Features

- **Hero Banner** - Eye-catching landing section with tagline
- **Product Grid** - 9 candy products displayed in a responsive card layout
- **Newsletter Signup** - Email subscription form to join the "Sweet Club"
- **Responsive Design** - Mobile-friendly layout that adapts to all screen sizes

## Products

- Cake Pops
- Rose Pop
- Icing Pop
- Gummy Bears
- Gummy Worms
- Peach Rings
- Lava Cups
- Valentine's Special
- Assorted Chocolates

## Tech Stack

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- [Font Awesome](https://fontawesome.com/) for icons
- [Google Fonts](https://fonts.google.com/) (Poppins)
